# Windows Server 2022 & AD Lab

## Présentation du projet
Ce projet est un lab pratique réalisé dans le cadre de mon apprentissage en informatique.  
Il a pour objectif de me familiariser avec Windows Server 2022 et les bases de l’Active Directory.

Le lab a été réalisé dans un environnement de machines virtuelles afin de reproduire un contexte proche du monde professionnel.

## Compétences développées
- Administration de base de Windows Server
- Active Directory (niveau débutant)
- Gestion des comptes utilisateurs
- Compréhension des environnements d’entreprise
- Méthodologie et documentation technique

## Environnement & outils
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- Machine virtuelle (VMware)
- Client Windows (pour tests de domaine)

## Configuration réalisée
- Installation de Windows Server 2022
- Promotion du serveur en **contrôleur de domaine**
- Création d’un domaine Active Directory
- Création et gestion :
  - d’utilisateurs
  - de groupes
- Réinitialisation de mots de passe
- Gestion des droits et permissions de base
- (Optionnel) Jonction d’un poste client au domaine
